# Style Guide

## C++
- File names: 
    - `.cc` for sources, `.hpp` for headers
    - `lower_snake_case.cc`
- class names: `class UpperCamelCase`
- function names: `void lowerCamelCase()`
- variable names: `int lowerCamelCase`
- namespace names: `namespace lower_snake_case`
